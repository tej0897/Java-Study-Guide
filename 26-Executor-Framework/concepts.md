# Day 26: Executor Framework (Thread Pools)

---

```text
ExecutorService - manages thread pool
Executors - factory class for creating executors

Types:
- FixedThreadPool - fixed number of threads
- CachedThreadPool - creates threads as needed
- SingleThreadExecutor - single thread
- ScheduledExecutor - scheduled execution

Benefits:
✓ Reuse threads
✓ Limit thread count
✓ Manage lifecycle
```
