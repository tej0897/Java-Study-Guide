# Day 29: Java Memory Model and Garbage Collection

---

```text
Memory Allocation:
- Heap: Objects
- Stack: Primitives, references
- PermGen/MetaSpace: Class metadata

Garbage Collection:
- Mark-and-Sweep
- Generational GC
- Young, Old, Permanent generations

Triggers: Memory pressure, explicit call

Best practices:
- Avoid memory leaks
- Use appropriate collection sizes
```
