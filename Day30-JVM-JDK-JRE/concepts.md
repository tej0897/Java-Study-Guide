# Day 30: JVM, JDK, JRE Deep Dive

---

```text
JVM (Java Virtual Machine):
- Executes bytecode
- Platform independent
- Memory management

JDK (Java Development Kit):
- Includes JVM + compiler + tools
- For development

JRE (Java Runtime Environment):
- Includes JVM + libraries
- For running applications

Bytecode: Compiled Java (.class files)
Compiled to machine code by JVM
```
